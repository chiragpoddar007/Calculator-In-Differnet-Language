# Calculator Language Comparison

A simple calculator implemented in three popular programming languages:

- Python
- C++
- Java

The goal of this project is to compare how the same program is written in different languages while keeping the functionality identical.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Invalid operator handling

## Languages Compared

| Language | File |
|----------|------|
| Python | `Python/calculator.py` |
| C++ | `CPP/calculator.cpp` |
| Java | `Java/Calculator.java` |

## Comparison

| Feature | Python | C++ | Java |
|---------|---------|------|------|
| Typing | Dynamic | Static | Static |
| Compilation | No | Yes | Yes |
| Beginner Friendly | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| Performance | High | Very High | High |
| Code Length | Short | Medium | Medium |

## Purpose

This repository is intended for:

- Students learning multiple programming languages
- Developers comparing syntax
- AI models analyzing equivalent implementations
- Anyone interested in understanding language differences through the same problem
